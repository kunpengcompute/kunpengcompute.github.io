ARM生态使能Landscape

  • Apache Hadoop
  • 分类: 大数据
  • Hadoop ARM CI已在社区正式支持，并于Hadoop 3.3.0版本发布首个ARM Release。
  • Apache Spark
  • 分类: 大数据
  • Spark是专为大规模数据处理而设计的快速通用的计算引擎。Spark ARM CI已经在社区正式启用。
  • Apache Hive
  • 分类: 大数据
  • Hive是一个基于Hadoop的数据仓库平台，Hive ARM CI已经在社区正式启用。
  • Apache HBase
  • 分类: 大数据
  • HBase是一个分布式的、面向列的开源数据库，HBase ARM CI已经在社区正式启用。
  • Apache Flink
  • 分类: 大数据
  • Flink是一个框架和分布式处理引擎，Flink ARM CI已经在社区正式启用。
  • Apache Kudu
  • 分类: 大数据
  • Kudu旨在完善Hadoop生态的存储层，用于对快速数据进行快速分析。
  • MariaDB
  • 分类: 数据库
  • MariaDB数据库管理系统是MySQL的一个分支，ARM CI已在社区正式启用。
  • Greenplum
  • 分类: 数据库
  • GreenPlum是面向数据仓库应用的关系型数据库，基于PostgreSQL开发。ARM CI已在社区正式启用。
  • TensorFlow
  • 分类: AI
  • TensorFlow是一个端到端开源机器学习平台，ARM Build CI已在社区每日构建。
  • PyTorch
  • 分类: AI
  • PyTorch是一个开源的Python机器学习库，ARM Build CI已在社区每日构建。
